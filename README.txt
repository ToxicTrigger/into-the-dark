플레이 방법 동일!
 1. 발판을 밟으면 스위치가 올라옵니다.
 2. 각 땅마다 발판이 1개씩 있습니다.
 3. 3개의 스위치를 모두 켜면 고대병기가 작동합니다.
 4. 고대병기가 작동하면 보스가 그로기 상태가 됩니다.


보스룸 메인 플레이
   발판 지형퍼즐

보스룸 페이즈 별 특징
   페이즈1 - 몬스터를 유인하는 요령 깨우치기
   페이즈2 - 화면에 잡히지 않는 먼 목표를, 토템 시스템을 이용하여 맞추기


테스트 바라는 부분
 1. 진행상의 버그
 2. 게임 오브젝트들의 개연성 
      (ex: 스위치를 켜면 고대병기가 작동되는 게 자연스러워 보이는가? 보스가 그로기 상태가 되는 게 자연스러워 보이는가?)
 3. 플레이에 대한 의견 
       (ex: 너무 어려워보인다, 어느 부분에는 설명이 반드시 필요해 보인다, 이 부분을 살리면 될 것 같다 등)
 4. 플레이를 살릴 연출 아이디어 (생긴다면...! ㅠㅠ)


주의사항
 1. 몬스터가 한 방에 모두 죽기 때문에 페이즈2는 클리어가 불가능한 상태입니다.
 2. 이 부분은 나중에 스테이지의 기본 시스템과 합치면 해결됩니다. (몬스터가 체력이 생기기 때문에)
 3. 따라서 페이즈2는 클리어를 했다 싶으실 때까지만 하시면 될 것 같습니다(ㅠㅠㅠ)
